# README

# Shiny Mango!

## Prerequisites
Ruby version 3 and up
Rails version 8 and up

## Starting dev environment

### Installing ruby
```bash
rbenv install 3.1.4
```
### Installing bundler
```bash
gem install bundler
```
### Install rails related gems and dependencies
```bash
bundle install
```

### Create databases (development and test dbs)
```bash
rails db:setup
```
